Centos 6 rpmbuild

Posted on 2019-03-23

Centos 6 自定义打包 rpm

需要的工具:
1.Centos 6
2.rpm-build 3.系统普通用户

yum install rpm-build

简单的打包过程

rpmbuild 简单参数说明:
rpmbuild [args] xxx.spec
-ba 既生成 src.rpm 又生成二进制 rpm
-bs 只生成 src 的 rpm
-bb 只生二进制的 rpm
-bp 执行到 pre
-bc 执行到 build 段
-bi 执行 install 段
-bl 检测有文件没包含

可以进行简单的测试先 rpmbuild -bp ,再-bc 再-bi 如果没问题再执行 -ba
